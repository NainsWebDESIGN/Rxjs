empty, never, throw 應用

 這幾個是比較無趣的 operators 但接下來會有很多實際使用的例子

 empty 有點像是數學上的零(0)，簡單來說會給我們一個空的Observable，訂閱他會直接送出complete的訊息

  
    import {{'{'}} Observable {{'}'}} from 'rxjs/Observable';
    import {{'{'}} empty {{'}'}} from 'rxjs/observable/empty';

    source: Observable{{'<'}}any{{'>'}} = empty();
      
    let Observer = {{'{'}}
      next: value => {{'{'}} console.log(value); {{'}'}},
      complete: () => {{'{'}} console.log('完成!'); {{'}'}},
      error: error => {{'{'}} console.log(error); {{'}'}}
    {{'}'}}
    this.source.subscribe(Observer);
    
    // 完成!

 never 會給我們一個無窮的 Observable ,訂閱他則什麼事情都不會發生，因為是發生在無窮久以後的事(兩條平行線永遠也無法完成...)

  
    import {{'{'}} Observable {{'}'}} from 'rxjs/Observable';
    import {{'{'}} never {{'}'}} from 'rxjs/observable/never';

    source: Observable{{'<'}}any{{'>'}} = never();
      
    let Observer = {{'{'}}
      next: value => {{'{'}} console.log(value); {{'}'}},
      complete: () => {{'{'}} console.log('完成!'); {{'}'}},
      error: error => {{'{'}} console.log(error); {{'}'}}
    {{'}'}}
    this.source.subscribe(Observer);

 throw 就只會拋出錯誤...
